# Spec: 홈 (/)

## 개요
사이트 진입점. 동문회 소개, 공지 미리보기, 빠른 메뉴를 제공한다.  
모든 사용자(비로그인 포함) 접근 가능.

---

## 권한

| 역할 | 접근 |
|------|------|
| 비로그인 | O (전체 읽기) |
| 로그인 동문 | O |
| 관리자 | O |

---

## 섹션 구성

### 1. 히어로 (Hero)
- 배경: 짙은 초록
- 뱃지: `YUHAN TECHNICAL HIGH SCHOOL ALUMNI`
- 제목: 유한공업고등학교 총동문회
- 부제: 유일한 박사의 숭고한 정신 문구
- CTA 버튼 2개: `동문 명부 보기 →` (`/alumni`), `공지사항` (`/notices`)

### 2. 설립정신
- 유일한 박사 소개 텍스트
- 3개 키워드 카드: 창립 정신 · 기술 연마 · 사회 봉사

### 3. 연혁 (History)
- 연도별 타임라인 리스트

### 4. 공지사항 미리보기
- 최신 공지 5개 제목 + 날짜
- `더보기 →` → `/notices`

### 5. 빠른 메뉴
- 8개 메뉴 아이콘 그리드 (공지사항, 자유게시판, 동문 명부, 행사 일정, 갤러리, 임원진, 장학회, 로그인/마이페이지)

---

## 데이터

- 공지사항 최신 5개: `GET /api/notices?limit=5`
- 정적 콘텐츠(설립정신, 연혁)는 하드코딩 또는 관리자가 수정 가능한 DB 필드
