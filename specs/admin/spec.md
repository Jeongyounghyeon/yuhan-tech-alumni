# Spec: 백오피스 (/admin)

## 개요
관리자 전용 백오피스. 모든 콘텐츠 CRUD 및 동문 승인/관리를 담당한다.  
ADMIN 권한 유저만 접근 가능.

---

## 권한

| 역할 | 접근 |
|------|------|
| 비로그인 / 동문 | X → `/` 리다이렉트 |
| 관리자 (ADMIN) | O |

---

## 라우트

| 경로 | 설명 |
|------|------|
| `/admin` | 대시보드 (통계 요약) |
| `/admin/members` | 동문 목록 + 승인/거절 |
| `/admin/notices` | 공지사항 관리 |
| `/admin/posts` | 자유게시판 관리 |
| `/admin/gallery` | 갤러리 관리 |
| `/admin/events` | 행사 일정 관리 |
| `/admin/officers` | 임원진 관리 |
| `/admin/scholarship` | 장학회 관리 |

---

## 기능별 상세

### 대시보드 (`/admin`)
- 요약 통계 카드: 전체 동문 수, 승인 대기 수, 공지 수, 갤러리 수
- 최근 가입 요청 목록 (빠른 승인/거절 버튼)

### 동문 관리 (`/admin/members`)
- 전체 동문 목록 (이름, 이메일, 상태, 가입일)
- 상태 필터: 전체 / PENDING / APPROVED / REJECTED
- 개별 승인 / 거절 버튼
- 상태 일괄 변경

### 공지사항 관리 (`/admin/notices`)
- 목록: 제목, 작성일, 조회수
- 작성 / 수정 / 삭제

### 자유게시판 관리 (`/admin/posts`)
- 목록: 제목, 작성자, 작성일, 댓글 수
- 삭제 (부적절 게시글 삭제)

### 갤러리 관리 (`/admin/gallery`)
- 목록: 제목, 이미지 수, 작성일
- 작성 / 수정 / 삭제

### 행사 관리 (`/admin/events`)
- 목록: 행사명, 날짜, 장소
- 등록 / 수정 / 삭제

### 임원진 관리 (`/admin/officers`)
- 목록: 이름, 직책, 순서
- 등록 / 수정 / 삭제
- 순서 변경 (order 필드)

### 장학회 관리 (`/admin/scholarship`)
- 소개 텍스트 수정
- 장학 목록: 등록 / 수정 / 삭제

---

## 레이아웃

- 좌측 사이드바 네비게이션 (데스크톱)
- 모바일: 상단 드로어 메뉴
- 공통 헤더: 관리자 이름, 로그아웃

---

## 수용 기준

- [ ] ADMIN 외 접근 시 `/`로 리다이렉트된다
- [ ] 동문 승인/거절이 즉시 반영되고 해당 유저의 세션 권한이 갱신된다
- [ ] 모든 CRUD 작업에 성공/실패 토스트 메시지가 표시된다
- [ ] 삭제 전 확인 다이얼로그가 표시된다
- [ ] 대시보드 통계가 실시간 DB 값을 반영한다
